# c语言
## 一. 概述
1. c语言的特点：简单、快速、高性能、兼容性好、功能强大、易于学习
2. c语言能干嘛：
    1. 小巧灵活、语法简单，适合做小工具。多个小工具一起甚至能形成操作系统，比如linux
    2. 硬件编程
    3. 对性能要求较高的应用程序。比如web服务器nginx（c语言写的），性能可以达到apache（C++写的）的10倍以上
