# 三 基础

## $router.go()
`this.$router.go({path : 'path' , query: { param: this.param} })`

`this.$router.go(0)`：可以刷新页面，但是有几个缺点
1. 页面会变白（一瞬间的白屏）
2. 不支持动态路由，即原本在某个入口添加进来的路由由于并没有再次执行该添加路由的方法，会导致原本已添加进来的动态路由丢失。
3. safari不支持

## $router.push()
用`this.$router.push()`重新路由到当前页面，页面是不会进行刷新的
