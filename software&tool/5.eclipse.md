## 三. 基础
### 3.1 关于work space和work set
1. space必须设置，每次eclipse启动就会把space的项目加载进来，一个space只能同时被一个eclipse打开
2. set相当于项目的分类，比如有公司的，有学习的，有其他人的，就可以用set来分类
3. 个人觉得space可以看做书，set看做目录
